# Minimal Blog Workspace Rules and Context

The following rules and context apply to all future conversations within this workspace.

## EnsoMori UI Design System and Aesthetics
This blog should adopt the "Zen", calm, and minimalist aesthetic of the EnsoMori Android app. Whenever updating styling, use the following core colors and principles:

### Dark Mode — Midnight Forest (Premium Dark Mode)
- **Background (Obsidian Ink):** `#080C0B` 
- **Surface (Deep Emerald Slate):** `#121A17`
- **Surface Variant:** `#1B2622` (Used for distinct elevations)
- **Primary / Accent (Frosted Emerald Glow):** `#9FFFD5`
- **Primary Container:** `#153329`
- **Text Primary (Pure Silk / Snow White):** `#F0F7F3`
- **Text Body / Secondary (Silver Mist):** `#B8C7C0`
- **Text Muted (Eucalyptus):** `#88A399`
- **Borders:** `#23302B`

### Light Mode — Soft Sage Green
- **Background:** `#F2F7F3`
- **Surface:** `#E8F2EB`
- **Surface Variant:** `#D0E8D6`
- **Primary (Deep Calm Forest Green):** `#2E7D52`
- **Primary Container:** `#B8DEC4`
- **Text Primary (Ink Deep):** `#0D2B18`
- **Text Muted:** `#3D6650`
- **Borders:** `#B0D3BB`

### Design Philosophy
- **Vibe:** Calm, intentional, "Zen".
- **Interactions:** Smooth with subtle micro-animations. Avoid generic generic, unpolished UI components. 
- **Typography:** Aim for a high-contrast, premium feel. Use `Recursive SemiBold` or a clean web-safe equivalent like `JetBrains Mono` or `Inter`.

## Brand Integration
- Always look for opportunities to feature EnsoMori cohesively, such as in the hero section, to establish a unified personal brand.
